# 😴 Ghid Sleep Cycle și Alarmy: Somn de calitate, dimineți productive

Somnul este **fundamentul** productivității, sănătății și bunăstării. Dar majoritatea oamenilor dorm prost și se trezesc morocănoși. Acest ghid explorează două aplicații complementare: **Sleep Cycle** pentru analiză și optimizare a somnului, și **Alarmy** pentru treziri care funcționează.

---

## 🌟 De ce contează somnul

### Impactul somnului de calitate

| Domeniu | Cu somn bun | Cu somn slab |
|---------|-------------|--------------|
| **Productivitate** | +25% | -50% |
| **Memorie** | Consolidare optimă | Afectată |
| **Imunitate** | Puternică | Slăbită |
| **Mood** | Stabil, pozitiv | Irabil, anxios |
| **Greutate** | Reglat | Tendință la îngrășare |

### Ciclu REM și Non-REM

Somnul are **4-6 cicluri** pe noapte (90 minute fiecare):

```
Ciclu:
├── Light Sleep (Non-REM 1-2)
├── Deep Sleep (Non-REM 3) — cel mai reparator
└── REM — visele, consolidarea memoriei

Trezirea în Light Sleep → te vei simți fresh
Trezirea în Deep Sleep → groggy pentru ore
```

## 🌙 Partea 1: Sleep Cycle — Analizantul somnului

### Ce face Sleep Cycle

📱 **Tracking:**
- Analizează mișcările din timpul nopții
- Detectează fazele de somn (light, deep, REM)
- Calculează un **Sleep Score** (0-100)

### Funcții principale

| Funcție | Descriere |
|---------|-----------|
| **Sleep Analysis** | Vezi cât ai dormit în fiecare fază |
| **Sleep Score** | Evaluare calitate 0-100 |
| **Smart Alarm** | Trezire în light sleep |
| **Snoring detection** | Detectează sforăitul |
| **Sleep Aid** | Sunete ambientale |
| **Trends** | Istoric pe săptămâni/luni |

### Smart Alarm — Trezire inteligenTă

**Problema:** Alarmele tradiționale te trezesc oricând — chiar și în deep sleep.

**Soluția Sleep Cycle:**
1. Setezi **fereastra de trezire** (ex: 6:30-7:00)
2. Sleep Cycle detectează când ești în **light sleep**
3. Te trezește în primul moment de light sleep din fereastră
4. Te trezești **natural**, fără grogginess

### Sleep Trends — Pattern-uri în timp

| Metric | Ce arată |
|--------|---------|
| Sleep Duration | Cât dormi efectiv |
| Sleep Quality | Scorul calității |
| Time in bed | Total time în pat |
| Deep Sleep | Somn profund (%)
| Sleep debt | Diferența față de necesar |

### Sfaturi Sleep Cycle

1. **Pune telefonul pe noptieră** — senzorul detectează mișcările
2. **Folosește constant** — minim 7 nopți pentru date accurate
3. **Verifică Trends săptămânal** — identifică pattern-uri

## ⏰ Partea 2: Alarmy — Alarmă care chiar funcționează

### De ce e diferit Alarmy

Alarmy este pentru cei care **nu pot dormi** după ce sună alarma:

| Problemă | Soluția Alarmy |
|----------|----------------|
| Apeși snooze | ❌ Snooze dezactivat |
| Dormi în continuare | ✅ Trebuie să faci task |
| Trezire automata nu ajunge | ✅ missions forcing |

### missions de trezire

Alarmy are **trei tipuri** de missions:

#### 1. Shake (Încărunță)
```
Setează: 30 de tremurături
☀️ Alarmă sună
❌ Nu poți dormi în continuare
```

#### 2. Math Problem (Matematică)
```
Setează: 5 probleme matematice, Hard difficulty
☀️ Alarmă sună
❌ Trebuie să rezolvi corect
```

#### 3. Photo (Fotografie)
```
Setează: Fa poza la chiuveta din baie
☀️ Alarmă sună
❌ Trebuie să te duci și pozezi
```

### Routine — Dimineața completă

Cu **Routine** poți programa:
```
7:00 - Alarmy
7:01 - Screenshot confirmare
7:05 - Deschide Spotify (muzică)
7:07 - Weather app
7:10 - Calendar view
```

### Sleep Talk Recorder

⚠️ **Funcție spion utilă:**
- Înregistrează ce spui în somn
- Dezvoltare: poate indica stres sau probleme
- **Atenție:** Confidențialitate vs beneficiu

## 🔄 Partea 3: Combo Strategy — Somn și trezire optimă

### Flow optim

```
🌙 Înainte de somn:
1. Set alarma în Sleep Cycle (Smart Alarm)
2. Set Alarmy pentru back-up
3. Pornește sunet ambiental (Rain, White noise)
4. Și Pune telefonul pe noptieră

⏰ Dimineața:
1. Sleep Cycle te trezește în light sleep (fresh)
2. [Backup] Alarmy mission dacă n-ai sărit din prima
```

### Optimizare somn

| Obicei | De ce contează |
|--------|---------------|
| **Ore fixe** | Reglează ritmul circadian |
| **Răcește camera** | 18-20°C ideal |
| **Evită ecranele 1h înainte** | Lumina albastră dereglează melatonină |
| **Cafea maxim până la 14:00** | Timpul de eliminare e lung |
| **Exercițiu zilnic** | Somn mai profund |
