# 🔍 Ghid Perplexity.ai: Search AI care răspunde cu surse

Perplexity.ai este un motor de căutare bazat pe AI care nu doar găsește link-uri, ci **răspunde direct la întrebări cu surse citate**. Este ca un research assistant care îți face munca de căutare și sinteză. Acest ghid te învață cum să-l folosești pentru cercetare eficientă.

## 🔧 Focus Modes — Moduri de căutare

Perplexity oferă moduri specializate pentru diferite tipuri de cercetare:

| Mod | Cel mai bun pentru |
|-----|-------------------|
| **Focus: Academic** | Articole științifice, studii |
| **Focus: Writing** | Content creation, copywriting |
| **Focus: YouTube** | Tutoriale video |
| **Focus: Reddit** | Opinii, discuții, experiențe |
| **Focus: Wolfram** | Calcule, date exacte |
| **All** | Căutare generalistă |

### Cum schimbi Focus Mode

1. În câmpul de căutare, click pe **„Focus"**
2. Selectează modul dorit
3. Sau tastează **"/academic"**, **"/youtube"**, etc.

## 📋 Partea 1: Căutare de bază

### Sintaxa de căutare

Perplexity înțelege întrebări naturale:

```
Întrebare: Care sunt cele mai bune practici pentru SEO în 2026?
Focus: All sau Writing

Întrebare: Ce spun studiile despre efectele cafeinei asupra somnului?
Focus: Academic

Întrebare: Tutoriale pentru learning Python de la zero
Focus: YouTube
```

### Structured Output

Răspunsurile vin structurate:

```
📌 Răspuns scurt
[Paragraf concis cu răspunsul principal]

📚 Surse
[1] titlu articol — link
[2] titlu articol — link

📖 Detalii
[Puncte cheie + explicații]
```

## 🎯 Partea 2: Research profund

### Thread-uri (Conversații)

Fiecare căutare creează un **Thread** — un fir de cercetare:

**Avantaje Thread:**
- Revino la cercetare oricând
- Continue din punctul unde ai rămas
- Adaugă întrebări follow-up

### Exemple de workflow de cercetare

**Cercetare pentru un articol:**

```
1. „Trend-uri în marketing digital 2026"
   → Vezi overview

2. „Detaliază despre AI în marketing"
   → Fii specific în follow-up

3. „Care sunt statisticile pentru email marketing?"
   → Solicită date concrete

4. „Găsește studii de caz de campanii de succes"
   → Dovezi și exemple
```

### Spatii — Colecții de thread-uri

**Spaces** organizează cercetările pe teme:

```
📁 Space: [Nume temă]
├── Thread: [Subtemă 1]
├── Thread: [Subtemă 2]
└── Thread: [Subtemă 3]
```

## 📊 Partea 3: Citează sursele corect

### Format citations

Perplexity citează automat:

```
[Sursa 1] — website.com
[Sursa 2] — revista.edu
[Sursa 3] — blog.gov
```

### Verifică sursele

⚠️ **Important:** Verifică întotdeauna sursele:
- Click pe link-uri pentru a confirma
- Preferă surse academice (.edu, .gov)
- Verifică data — info recentă > veche
- Cross-reference cu alte surse

### Caută doar surse de încredere

```
site:.edu OR site:.gov [întrebare]
```

Aceasta limitează rezultatele la surse academice și guvernamentale.
