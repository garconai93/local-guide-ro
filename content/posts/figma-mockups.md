# 🎨 Cum să creezi mockups simple cu Figma (free)

Figma a devenit instrumentul de design colaborativ cel mai popular din lume, fiind folosit de echipe de la Airbnb, Spotify, Uber și mii de alte companii. Și cel mai bun? **Figma oferă un plan gratuit extrem de generos** — poți crea mockups profesioniste, wireframes și prototipuri interactive, totul fără să plătești un leu. În acest ghid complet, vei învăța cum să folosești Figma de la zero pentru a crea mockups simple dar eficiente.

### Variante de utilizare

| Variantă | De ce să o alegi |
|---------|-----------------|
| **Browser (figma.com)** | Funcționează imediat, nu instalezi nimic |
| **Desktop App (Mac/Windows)** | Mai rapid, funcționează offline |

### Setup pentru începători

1. Accesează [figma.com](https://figma.com) și creează un cont
2. Verifică email-ul și confirmă contul
3. Creează primul proiect: **File → New file**

## 📐 Crearea unui mockup — Tutorial pas cu pas

### Proiect: Wireframe pentru o aplicație mobile

### Pasul 1: Creează un Frame pentru ecran

1. Apasă **F** sau selectează Frame din toolbar
2. În panoul din dreapta, alege **iPhone 14 Pro** (sau alt device)
3. Frame-ul apare pe canvas

### Pasul 2: Adaugă Header-ul

1. Selectează **Rectangle tool (R)**
2. Desenează un dreptunghi în partea de sus (390 x 60 px)
3. Setează culoarea: alb sau gri deschis (#F5F5F5)
4. Adaugă text: „App Name" — aliniază centrat

### Pasul 3: Creează o imagine placeholder

1. **R** → desenează un dreptunghi pentru banner
2. Click pe **Fill** în dreapta → alege **Color** → gri
3. Pentru a adăuga icon, folosește **plugin-ul Content** (gratuț):
   - Plugins → Content → Placeholder image
   - Alege tipul: „Image"

### Pasul 4: Liste și butoane

**Lista de articole:**
1. Desenează 3 dreptunghiuri (card-uri)
2. Fiecare card are: imagineă + text (titlu + descriere)
3. Grupează elementele: **Cmd/Ctrl + G**

**Butoane:**
1. **R** → desenează un buton
2. Corner radius: **8** (colțuri rotunde)
3. Fill: albastru (#3B82F6)
4. Text: „Button" centrat, alb

## 🧩 Componente — Reutilizare inteligentă

### Ce sunt componentele?

Componentele sunt elemente reutilizabile. Când modifici o componentă, toate instantele se actualizează.

### Crearea unei componente

1. Desenează un buton cu toate proprietățile
2. Selectează-l → **Cmd/Ctrl + K** (Create component)
3. Butonul apare în Assets panel (stânga) și în sidebar

### Folosirea componentelor

1. Drag & drop din Assets panel pe canvas
2. Fiecare copie este o „Instance"
3. Pentru a edita o instanță specifică: **Cmd/Ctrl + Alt + K** (Detach)

### Variante de componente

Pentru butoane cu mai multe stări:

1. Selectează componenta → **Cmd/Ctrl + Alt + B** (Add variant)
2. Adaugă variante: Primary, Secondary, Ghost
3. Adaugă proprietăți: Size (Small, Medium, Large)

## 🔗 Prototyping — Adaugă interactivitate

### Setup prototyping

1. Click pe tab-ul **Prototype** în dreapta
2. Sau tastează **Cmd/Ctrl + E** (echivalent)
3. Interfața se schimbă pentru a arăta conexiunile

### Crearea unui flow simplu

**Exemplu: Ecran de login → Home**

1. **Ecran Login:**
   - Frame: 390 x 844 (iPhone)
   - Include: Email input, Password input, Login button

2. **Ecran Home:**
   - Alt frame cu UI-ul principal
   - Include: Header, banner, listă produse

3. **Conectarea ecranelor:**
   - Click pe butonul „Login"
   - Trage săgeata către ecranul Home
   - În pop-up, setează:
     - **Interaction:** On Click
     - **Destination:** Home Screen
     - **Animation:** Dissolve (300ms)

### Opțiuni de tranziție

| Tranziție | Utilizare |
|-----------|-----------|
| **Dissolve** | Fade smooth între ecrane |
| **Smart Animate** | Animații automate bazate pe layer |
| **Move In/Out** | Slide din lateral |
| **Push** | Ca o navigare nativă |
| **Cover** | Acoperă ecranul anterior |

## 📱 Export — Cum să exporti design-ul

### Metode de export

| Metodă | Pași |
|--------|------|
| **Selection** | Selectează elementele → Cmd/Ctrl + Shift + E |
| **Full frame** | Click dreapta pe frame → Export |
| **Bulk export** | Selectează multiple elemente |

### Formate disponibile

| Format | Utilizare |
|--------|-----------|
| **PNG** | Imagini cu transparență, web |
| **JPG** | Fotografii, background-uri |
| **SVG** | Vectorial, scalabil, code |
| **PDF** | Pentru print sau documente |

### Export 1x, 2x, 3x

Pentru retina/HD:

1. În fereastra de export, adaugă scale:
   - **1x** → Telefoane vechi
   - **2x** → iPhone standard
   - **3x** → iPhone Pro Max, flagship Android

## 💡 Sfaturi pentru mockups eficiente

### Best practices

| ❌ Greșit | ✅ Corect |
|---------|---------|
| Prea multe culori | Max 3-4 culori principale |
| Text greu de citit | Contrast ridicat, font size min 14px |
| Elemente nealiniate | Folosește Grid și Snap to Grid |
| Overdesign | Începe cu wireframe grayscale |
| Ignori spacing | 8px grid system |

### Structura unui proiect bun

```
📁 Proiect: App Mobile Redesign
├── 📄 Cover (prezentare)
├── 📄 Flows
│   ├── 📱 Login Flow
│   ├── 📱 Onboarding Flow
│   └── 📱 Home Flow
├── 📄 Components
│   ├── Buttons
│   ├── Cards
│   └── Navigation
├── 📄 Screens
│   ├── Login
│   ├── Home
│   └── Profile
└── 📄 Assets
    ├── Icons
    └── Images
```

### Auto Layout — Responsivitate ușoară

Auto Layout permite container-elor să se redimensioneze automat:

1. Selectează un grup sau frame
2. Click pe iconița Auto Layout în dreapta
3. Setează:
   - **Direction:** Horizontal sau Vertical
   - **Spacing:** Distanța între elemente
   - **Padding:** Spațiul interior
