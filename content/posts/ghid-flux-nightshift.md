# 👁️ Ghid F.lux și Night Shift: Sănătatea ochilor și somnul

Lumina albastră de pe ecrane este **dușmanul tăcut** al somnului și al ochilor. Lumina albastră suprima melatonina (hormonul somnului), cauzează oboseală digitală a ochilor și poate afecta retina pe termen lung. Acest ghid îți arată cum să contracarezi cu **F.lux** și **Night Shift**.

---

## 🔵 Partea 1: Ce este lumina albastră și de ce contează

### Ce este lumina albastră

Lumina albastră are **lungime de undă scurtă** (400-490nm) și:
- Este emisă puternic de ecrane LED, LCD
- Pătrunde până la retină
- Suprima producția de **melatonină**

### Impactul luminii albastre

| Moment | Efect |
|--------|-------|
| **Ziua** | Lumina albastră = alertă, concentrare |
| **Seara** | Suprima somnolența naturală |
| **Noaptea** | Întârzie somnul cu 1-2 ore |

### Oboseala digitală a ochilor (Digital Eye Strain)

Simptome după 2+ ore de ecran:
- Ochii uscați
- Vedere neclară temporar
- Dureri de cap
- Dificultate de focus

## 🌅 Partea 2: Night Shift — Soluția Apple

Night Shift este **integrat în Apple** (iOS, iPadOS, macOS):

### Activare automată

```
iPhone/iPad:
Setări → Afișare și Strălucire → Night Shift
→ Program: Sunset to Sunrise
→ Activat ✅

Mac:
System Preferences → Displays → Night Shift
→ Program: Sunset to Sunrise
→ Activat ✅
```

### Setări manuale

| Setare | Valoare recomandată |
|--------|---------------------|
| **Color Temperature** | 10-20% mai cald |
| **Program** | Sunset to Sunrise |
| **Manual** | Pornește seara la 20:00 |

### Ce face Night Shift

- Schimbă balansul de culoare spre **galben/portocaliu**
- Reduce lumina albastră
- Se activează și dezactivează automat

### Limitări Night Shift

❌ Nu este la fel de puternic ca F.lux
❌ Nu are tranziție graduală
❌ Setări limitate

## 💡 Partea 3: F.lux — Soluția completă pentru Windows/Mac

F.lux este **mai puternic și mai configurabil**:

### Descarcă și instalează

1. [justgetflux.com](https://justgetflux.com)
2. Alege sistemul tău (Windows/Mac/Linux)
3. Instalează și restart

### Setup inițial

```
1. F.lux pornește automat
2. Click pe icon în taskbar/tray
3. Setări:
   - Latitude/Longitude (sau oraș) — pentru sunset timing
   - Wake up time — pentru tranziția de dimineață
```

### Pre-set-uri de culoare

| Mod | Când | Temperatura |
|-----|------|-------------|
| **Daytime** | 9:00 - 17:00 | 6500K (natural) |
| **Sunset** | 17:00 - 20:00 | 4500K (warm) |
| **Night** | 20:00 - 8:00 | 3400K (orange) |
| **Sleep** | După ora de culcare | 2700K (dim) |

### Tranziții suave

F.lux schimbă temperatura **gradual**:
```
18:00 — Începe tranziția (6500K → 5000K)
20:00 — Continuă (5000K → 4000K)
21:00 — Aproape de Night (4000K → 3400K)
22:00 — Mod Sleep complet (3400K → 2700K)
```

### Funcții avansate

| Funcție | Descriere |
|---------|-----------|
| **Movie mode** | Temporar 6500K pentru design |
| **Darkroom mode** | Roșu foarte cald (pentru fotosensibilitate) |
| **Disable for app** | Exclude aplicații (design, foto) |
| **Hotkeys** | Toggle rapid |

### F.lux vs No F.lux

```
Fără F.lux noaptea:
☀️ Ecran albastru intens
😴 Creierul crede că e zi
🛏️ Somn întârziat

Cu F.lux noaptea:
🌙 Ecran portocaliu cald
😌 Creierul pregătit pentru somn
🛏️ Somn natural
```

## 🌙 Partea 4: Best Practices

### Rutina de seară

```
18:00 — F.lux începe tranziția
20:00 — Pornește Night Shift (dacă nu e auto)
21:00 — Redu intensitatea ecranului
22:00 — Mod Sleep complet
```

### Setări recomandate

| Platformă | Temperatura noapte | Motivație |
|-----------|-------------------|-----------|
| **Mac** | 3500-4000K | F.lux |
| **Windows** | 3400-3700K | F.lux |
| **iPhone** | Warmest | Night Shift |
| **iPad** | Warmest | Night Shift |

### Exclude aplicații când e necesar

Când lucrezi cu culori (design, foto, video):
1. F.lux → Movie mode
2. Night Shift → Off temporar
3. Sau exclude app din F.lux settings
