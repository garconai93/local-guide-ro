# ✅ Ghid Todoist: Sistemul tău de task-uri pentru o productivitate absolută

Todoist este unul dintre cele mai populare și apreciate aplicații de task management din lume. Cu o interfață minimală dar extrem de puternică, Todoist te ajută să capeți controlul asupra sarcinilor zilnice și să nu mai uiți nimic important. Acest ghid te transformă din începător în utilizator avansat.

---

## 🌟 De ce Todoist?

**Todoist** este preferat de milioane de utilizatori datorită:

- **Simplitate** — interfață curată, fără clutter
- **Cross-platform** — funcționează pe web, iOS, Android, Mac, Windows
- **Colaborativ** — poți lucra la proiecte cu echipa
- **Natural Language** — adaugi task-uri rapid: „Întâlnire mâine la 10"
- **Integrare** — se conectează cu sute de aplicații

---

## 📝 Bazele: Cum adaugi task-uri

### Metoda 1: Natural Language (cel mai rapid)

În loc să umbli prin meniuri, tastează direct:

```
Întâlnire cu echipa azi la 15:00 #Munca
Cumpără lapte până vineri #Personal
Raport final! p1 mâine #Munca
```

**Ce înțelege Todoist:**
- **„azi", „mâine", „săptămâna viitoare"** → date automate
- **#NumeProiect** → adaugă la proiect
- **p1, p2, p3, p4** → prioritate (1 = cea mai înaltă)
- **!+** → etichetă de labels

### Metoda 2: Click și completare

Pentru task-uri complexe:
1. Apasă **Q** (scurtătură rapidă) sau click pe **+**
2. Completează titlul, proiectul, data, prioritatea
3. Adaugă descrieri sau comentarii
4. Bifează **„Adaugă fișiere"** pentru atașamente

## 🏗️ Proiecte și structură

### Ierarhia Todoist

```
📂 Proiect Principal (Poate conține subproiecte)
├── 📁 Subproiect 1
│   └── 📋 Task-uri
├── 📁 Subproiect 2
│   └── 📋 Task-uri
```

### Proiecte recomandate pentru start

| Proiect | Descriere |
|---------|-----------|
| 🏠 Acasă | Sarcini casnice, reparații |
| 💼 Muncă | Task-uri profesionale |
| 📚 Învățare | Cursuri, lectură, certificări |
| 💰 Finanțe | Budget, facturi, investiții |
| 🏋️ Sănătate | Sport, medic, alimentație |
| 📅 Evenimente | Petreceri, călătorii, nunți |

### Best practices pentru proiecte

- **Maxim 5-7 proiecte principale** — altfel devine overwhelming
- **Subproiecte pentru complexitate** — împarte proiectele mari în pași mici
- **Denumiri scurte și clare** — fără „Proiectul Super Important #123"

## 🔢 Priorități — GTD-style

### Sistemul de priorități Todoist

| Simbol | Nivel | Când să-l folosești |
|--------|-------|-------------------|
| p1 | Urgent + Important | Deadline azi, crize |
| p2 | Important, nu urgent | Deadline săptămâna asta |
| p3 | Dezirabil | Faceți când ai timp |
| p4 | Delega | Poate chiar să nu faci |

### Regula lui Eisenhower

Combină prioritățile cu categorii:

```
p1 = Urgent + Important (FĂ ACUM)
p2 = Important, nu urgent (PROGRAMEAZĂ)
p3 = Urgent, nu important (DELEGA)
p4 = Nici urgent, nici important (ELIMINĂ)
```

## 📅 Date și deadline-uri

### Sintaxă pentru date rapide

- `azi` — astăzi
- `mâine` — mâine
- `luni`, `marți`, `miercuri`... — ziua din săptămână
- `1/4` — 1 aprilie (anul curent sau viitor)
- `overmorrow` — poimâine
- `eod` — end of day (la miezul nopții)

### Recurența — Automatizează repetiția

Pentru task-uri repetitive:

```
În fiecare zi la 8:00 → rec: zilnic
În fiecare luni → rec: săptămânal
Prima zi a fiecărei luni → rec: lunar
În fiecare 2 săptămâni → rec: every 2 weeks
```

**Exemplu practic:**
```
Periaj dentar! every day
Taxe și impozite every year on jan 15
```

## 🏷️ Etichete (Labels) — organizare transversală

Etichetele în Todoist funcționează diferit de proiecte: **un task poate avea mai multe etichete** și le poi filtra independent de proiect.

### Etichete populare

| Etichetă | Culoare | Pentru ce |
|----------|---------|-----------|
| @context/birou | Albastru | Task-uri care necesită să fii la birou |
| @context/online | Verde | Task-uri care necesită internet |
| @context/telefon | Portocaliu | Apeluri, conversații |
| @delegat | Roșu | Task-uri delegate altora |
| @ waiting | Galben | Aștepți răspuns/acțiune |

### Folosește etichetele pentru context

În loc să filtrați prin proiecte, filtrați după **unde sunteți** sau **cu ce echipament**:

```
@context/calculator → toate task-urile care necesită PC
@context/telefon → apeluri de făcut
@online → task-uri care necesită internet
```

## 🔍 Filtre și Interogări

Filtrele sunt ca interogări salvate care arată task-uri din mai multe proiecte simultan.

### Filtre built-in utile

- **Azii** — toate task-urile pentru astăzi
- **Următoarele 7 zile** — planificare pe termen scurt
- **Priorități** — doar p1 și p2
- **Fără dată** — task-uri fără deadline

### Creează filtre personalizate

Exemplu: „Task-uri mele de azi, fără delegări"
```
(due: today & !@delegat & assigned to: me) | overdue
```

### Filtre salvate frecvent

| Filtru | Interogare |
|--------|-----------|
| Busy days | `due: today & p1` |
| Waiting for | `@waiting` |
| This week focus | `due before: saturday & p1\|p2` |
| Inbox zero | `#Inbox` |

## 👥 Colaborare în Todoist

### Invită membri la proiect

1. Deschide proiectul → **Share**
2. Introdu adresa de email
3. Alege permisiunile: **Can edit** sau **Can view**

###Comentarii și notificări

- **@mention** în comentarii pentru a atrage atenția cuiva
- Sistemul trimite notificări automate
- Toată conversația rămâne în contextul task-ului
