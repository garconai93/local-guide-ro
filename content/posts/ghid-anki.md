# 🧠 Ghid Anki: Flashcards inteligente pentru învățare rapidă

Anki este un program de flashcards bazat pe **repetiție spațiată (Spaced Repetition)** — o tehnică dovedită științific care te ajută să-ți amintești informații pentru totdeauna. Indiferent dacă înveți o limbă străină, pregătești un examen sau memorezi definiții, Anki este cel mai puternic instrument. Acest ghid te învață cum să-l folosești corect.

---

## 🌟 Ce este Repetiția Spațiată?

### Știința din spate

Creierul nostru uită informații în timp — dar cu **revizuire în momentul optim**, consolidează memoria pe termen lung.

```
Prima învățare → Uitare rapidă (ore)
Revizuire la 1 zi → Uitare mai lentă (zile)
Revizuire la 3 zile → Uitare și mai lent (săptămâni)
Revizuire la 7 zile → Memorie pe termen lung (luni)
Revizuire la 30+ zile → Memorie permanentă
```

### Algoritmul Anki

Anki folosește un algoritm care:
- **Monitorizează** cât de bine ai răspuns
- **Calculează** când vei uita
- **Programează** revizuirea exactă când ai nevoie
- **Optimizează** timpul de studiu

## 📝 Partea 2: Crearea Flashcards-urilor eficiente

### Tipuri de card-uri

#### 1. Basic (Simplu)
```
Front: Ce înseamnă "hello" în engleză?
Back: Hello = Bună ziua
```

#### 2. Basic (Reversed)
Anki generează automat varianta inversă:
```
Front: Bună ziua
Back: Hello
```

#### 3. Cloze Deletion (Text cu gol)
```
Paris este capitala {{c1}} → [Show Answer] → Franței
```

#### 4. Image Occlusion
Acoperă o parte dintr-o imagine:
```
┌──────────────┐
│ Diagramă     │ → Selectează zona → Ascunde
│ Biologie     │
└──────────────┘
```

### Reguli pentru flashcards bune

| Regulă | De ce contează |
|--------|---------------|
| **O întrebare per card** | Mai ușor de procesat |
| **Răspuns scurt** | Retenție mai bună |
| **Contexte** | Cuvintele în propoziții se memorează mai bine |
| **Vizual > Text** | Imaginile lasă amprente mai puternice |
| **Nu învăța doar definiții** | Includ examples, nuances |

### Exemple practice

**Pentru limbi (Engleză → Română):**
```
Front: ephemeral
Back: efemer, de scurtă durată, trecător
      "The beauty of cherry blossoms is ephemeral"
```

**Pentru medicină:**
```
Front: Care sunt simptomele diabetului tip 2?
Back: • Sete excesivă (polidipsie)
      • Urinare frecventă (poliurie)
      • Foame constantă
      • Oboseală
      • Vedere neclară
```

```
│   ├── 📁 Engleză

│   │   ├── 📋 Expresii

│   └── 📁 Franceză
│       └── 📋 Începători
├── 📁 Medicină
│   ├── 📋 Anatomie
│   └── 📋 Farmacologie
├── 📁 Istorie
│   └── 📋 România
└── 📁 Programare
    └── 📋 Python
```

### Tags vs Decks

| Abordare | Când folosești |
|----------|---------------|
| **Decks separate** | Subiecte total diferite |
| **Tags** | Cross-reference (același card apare la mai multe categorii) |

## ⏰ Partea 4: Rutina zilnică de studiu

### Ideal Daily Flow

```
🌅 Dimineața (20 min)
└── Anki — review cards

📚 Seara (30 min)
├── Anki — learn new cards
└── Anki — review cards

📅 Săptămânal (1-2 ore)
├── Adaugă cards noi
├── Curăță cards vechi
└── Evaluate progress
```

### Câte card-uri noi pe zi?

| Obiectiv | Card-uri noi/zi | Timp zilnic |
|---------|----------------|-------------|
| Hobby | 5-10 | 15 min |
| Student | 20-30 | 30-45 min |
| Intensiv (traducător) | 50-100 | 1-2 ore |

### Regula de aur

⚠️ **Calitatea > Cantitatea**

Mai bine 10 cards bine făcute decât 50 cards slabe!

## 🔧 Partea 5: Funcții avansate

### 1. Add-ons utile

| Add-on | Ce face |
|--------|--------|
| **Heatmap** | Arată streak-ul zilnic |
| **Advance Scheduler** | Mai multe opțiuni de scheduling |
| **Frozen Fields** | Păstrează câmpuri când adaugi multiple |
| **Browse Gallery** | Vizualizează cards ca grid |

### 2. Suspend și Bury

- **Suspend** — exclude temporar card-ul din review
- **Bury** — amână card-ul pentru mâine

### 3. Filtered Decks

Creează decks temporare pentru:
- Cards pe care le-ai ratat
- Cards din ultimele 7 zile
- Cards cu tag specific

### 4. Statistics

Anki oferă statistici detaliate:
- Cards revizuite
- Rata de succes
- Timp mediu de răspuns
- Streak zilnic

## 💡 Sfaturi finale pentru succes

1. **Începe cu 5-10 cards noi/zi** — prea multe = burnout
2. **Fă-ți un habit, nu un task** — zilnic, nu când ai timp
3. **Cards simple > complexe** — un concept per card
4. **Revizuiește înainte de a dormi** — consolidează memoria
5. **Nu sări peste review** — consistency > intensity
6. **Crează cards contextuale** — nu doar definiții izolate
